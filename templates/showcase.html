Backend Architecture

  • Flask REST API with session-based auth
  • OTP verification (email & phone)
  • MusicGen loaded once on startup
  • SQLite via helper functions in database.py
  • Gunicorn WSGI server for production
  • Environment-driven secrets (GROQ_API_KEY)

Frontend Architecture

  • Vanilla JS — no framework overhead
  • WaveSurfer.js for real-time waveform display
  • Responsive CSS with dark/light mode toggle
  • Feature tabs: Music, Lyrics, Remix, Video BGM
  • Live prompt enhancement via Groq API
  • Instrument & mixer sliders for fine-tuning
